# DropQuiz
このゲームは正解の床を選ぶだけのシンプルなクイズゲームです。

# 操作方法
| Key |  Action  |
| ---- | ---- |
|  WASD  |  プレイヤーの移動 |
|  IJKL  |  カメラの回転 |
